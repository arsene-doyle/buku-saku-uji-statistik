Konsep Dasar Statistik Biomedik

Pahami 4 pilar ini sebelum menggunakan alur keputusan.

1. Skala Pengukuran (NOIR) ▼

Skala pengukuran menentukan jenis uji yang dapat digunakan. Dikelompokkan menjadi dua besar: Kategorik dan Numerik.

Kelompok Skala Sifat Contoh Data Klinis
Kategorik Nominal Hanya membedakan, tidak ada urutan Jenis kelamin (L/P), Golongan darah (A/B/AB/O), Status merokok (Ya/Tidak)
Ordinal Ada tingkatan/urutan yang bermakna Derajat nyeri (Ringan/Sedang/Berat), Stadium Kanker (I-IV), Skala Likert
Numerik Interval Ada jarak pasti, nilai 0 mutlak tidak ada Suhu tubuh (°C), Skor IQ, Skor HARS
Rasio Ada kelipatan, memiliki nilai 0 mutlak Berat badan (kg), Tekanan darah (mmHg), Kadar Hb (g/dL), Jumlah koloni bakteri
2. Tujuan Penelitian ▼
Uji Beda (Komparasi)

Fokus: Membandingkan nilai antar kelompok.

Kata kunci di judul: "Perbedaan", "Perbandingan", "Efektivitas".

Contoh: Apakah tekanan darah kelompok intervensi berbeda dari kontrol?

Uji Hubungan (Korelasi)

Fokus: Melihat kaitan antara dua variabel sejajar.

Kata kunci di judul: "Hubungan", "Korelasi", "Asosiasi".

Contoh: Apakah ada hubungan antara status merokok dan kejadian asma?

Uji Prediksi (Regresi)

Fokus: Memprediksi variabel dependen berdasarkan independen.

Kata kunci di judul: "Pengaruh", "Prediksi", "Faktor Risiko", "Model".

Contoh: Apakah IMT dapat memprediksi tekanan darah sistolik?

3. Berpasangan vs Independen ▼
Berpasangan (Paired)
Independen (Unpaired)
Kriteria Berpasangan Independen
Subjek Orang yang sama (atau sengaja dipasangkan/matching) Orang yang berbeda di tiap kelompok
Desain Khas Pre-test & Post-test, Cross-over Kasus-Kontrol, Intervensi vs Plasebo
Kesalahan umum: Mengukur Pre dan Post pada SATU kelompok perlakuan merupakan analisis BERPASANGAN, bukan membandingkan dua kelompok independen.
4. Distribusi Normal ▼

Syarat utama uji parametrik (untuk data numerik) adalah distribusi data normal.

Langkah SPSS Uji Normalitas:

  1. Klik Analyze > Descriptive Statistics > Explore...
  2. Masukkan variabel numerik ke kotak Dependent List.
  3. Klik tombol Plots...
  4. Centang Normality plots with tests, lalu klik Continue dan OK.

Cara Baca Output (Kolom Shapiro-Wilk untuk n < 50 atau Kolmogorov-Smirnov untuk n ≥ 50):

  • Nilai Sig. (p-value) > 0,05 = Data berdistribusi Normal → Gunakan Uji Parametrik
  • Nilai Sig. (p-value) ≤ 0,05 = Data Tidak Normal → Gunakan Uji Non-Parametrik
